Några lärdomar från ovanstående övningar... och från andra i läroboken

  • Det ska vara samma värde på "for" och "id" (Belfrage, Webbutveckling 1, s. 56)
  • Placeholder - innehåller text som syns i en formulärruta. Denna försvinner när man börjar skriva text.
  • Placeholder syns inte om man i stället anger "Kommentar:" som en del av Innehåll.
  • Exempel:
    input type="text" name="namn" value="Andrea"
    input type="number" name="age" value="15"
    input type="email" name="mail" value="example@mail.com"
    Dessa skickas som:
    namn=Andrea
    age=15
    mail=example@mail.com
  • För tillgänglighetens skull behöver label vara kopplad till input. Detta kan göras antingen genom nästling (att input placeras inuti label), eller genom att label använder attributet for med samma värde som input-elementets id.
    När man använder "laber for" + "id"
  • En knapp kan hjälpa användaren att interagera med webbsidan utan att skicka in datan. Med andra ord kan en knapp kopplas ihop med en händelse. Till exempel kan användaren ändra bakgrundsfärg när hen trycker på en knapp (Belfrage, s. 61). Bakgrundsfärgen ändras då genom JavaScript kod.
